🌦️ App de Clima - Módulo 4
Temática: Ruta de Destinos Naturales.

📊 Lógica de Datos
- Modelo: Los datos se gestionan en un arreglo de objetos `lugares` en JS, eliminando contenido estático del HTML.
- Cálculos: Se utilizan funciones con ciclos `for` para obtener temperaturas mínimas, máximas y promedios semanales.
- Resumen: Mediante condicionales, la app genera un mensaje textual basado en la cantidad de días soleados o lluviosos.
